手动ubuntu 18.04修改登录锁屏界面效果(含登录背景修改)flat-remix

前言

在ubuntu 18.04，可以通过修改/etc/alternatives/gdm3.css来进行修改
本来想直接使用flat-remix主题，但是只有这个登录界面没有达到作者演示的效果，所以手动覆盖修改
flat-remix主题

修改开始

1首先，进行备份,这个可以在错误时进行恢复，这个很重要

sudo cp /etc/alternatives/gdm3.css /etc/alternatives/gdm3_bak.css

2如果只需要修改登录背景

sudo gedit /etc/alternatives/gdm3.css

找到lockDialogGroup

#lockDialogGroup {
  background: #2c001e url(在这里填写你想要的图片);         
  background-repeat: no-repeat;
  background-size: cover;
  background-position: center; 
}

url格式如下:

file:///usr/share/backgrounds/login-background2
