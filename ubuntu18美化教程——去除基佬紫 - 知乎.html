开机之后首先进入的GRUB2 2.02

首先对不了解grub的新手介绍下，在早期的GNU/LINUX系统中，使用 Grub 作为系统的启动引导程序，想修改系统启动项非常简单，只要打开系统菜单设定文件 /boot/grub/menu.lst，修改该文件中的内容即可，但是到了 Grub2，可就不能采用这种方式修改系统启动项了。Grub2（GRand Unified Bootloader, version 2）是 Grub 的第二版。

Grub 2 包含下面几个部分：

|
|-- /boot/grub/grub.cfg 文件
|-- /etc/grub.d/ 文件夹
|-- /etc/default/grub 文件

/boot/grub/grub.cfg的文件是运行update-grub自动生成的自动生成，所以更改这个文件中的内容没有意义。要修改配置文件的只要打开/boot/grub/grub.cfg文件，找到想修改的地方，然后根据注释找到相应的 /etc/default/grub或/etc/grub.d/ (folder)进行修改。

介绍下/etc/grub.d/的文件：

.
├── 00_header          
├── 05_debian_theme             grub2加载页面主题
├── 10_linux                    10_linux为系统自动添加的当前root分区linux引导项
├── 20_linux_xen
├── 20_memtest86+               20_memtest86+为系统自动添加的内存测试菜单项
├── 30_os-prober                对以ntldr引导的系统如win2000，xp，win2003，因其引导机制只能从第一硬盘启动
├── 30_uefi-firmware
├── 40_custom                   40_custom为自定义的启动项，如启动cdlinux
├── 41_custom
└── README

grub2美化方案：

1 修改背景图和文字颜色

Grub2/Displays - Community Help Wiki​help.ubuntu.com

修改背景基佬紫图片，把你的图片直接放到这个目录下：

/boot/grub

修改文字颜色和文字背景框：

在 /etc/grub.d/05_debian_theme中找到下行代码：

if [ -z "${2}" ] && [ -z "${3}" ]; then
    echo "  true"
fi

增加

if [ -z "${2}" ] && [ -z "${3}" ]; then
    # echo "  true"
    echo "    set color_highlight=color1/color2"
    echo "    set color_normal=color1/black"
fi
语句介绍：
menu_color_highlight=text-color/bg-color
#The color of the highlighted menuentry and its background within the menu 'box'.
menu_color_normal=text-color/bg-color
#The color of non-selected menuentries within the menu 'box'
color_normal=text-color/bg-color
#The color of the text and background outside the menu 'box'

修改引导顺序：

方案一

Grub2/Submenus - Community Help Wiki​help.ubuntu.com

方案二 在/etc/grub.d/40_custom中自定义启动项：

menuentry "启动 Veket"{  
set root=(hd0,8)  
linux /veket/vmlinuz root=/dev/ram0 PMEDIA=hd  
initrd /veket/initrd.gz  
}  
menuentry "启动 CDLinux"{  
set root=(hd0,8)  
linux /CDlinux/bzImage root=/dev/ram0 vga=791 CDL_LANG=zh_CN.UTF-8  
initrd /CDlinux/initrd  
}  
menuentry "启动 Grub4Dos"{  
set root=(hd0,8)  
linux /grub.exe  
}

我用的不是这么简单的上下顺序修改，过程比较复杂，可以在GRUB2 文档中仔细看看。

全部改好以后记得update-grub，你不把你改的文件加载好，开启还是原来的样子呢。

开机之后加载的ubuntu加载图

系统加载过程中有一个紫色的背景和ubuntu加载图标，怎么修改这个？

首先要安装包plymouth-themes

sudo apt install plymouth-themes

把你要安装的主题放到这个目录下/lib/plymouth/themes

修改这个目录下的配置：

/usr/share/plymouth/themes/default.plymouth

为你安装的目录下的 ‘你安装的配置名字’.plymouth

更换背景图片的方式是一样的，把default.plymouth配置下的目录中的.png图片换成你想要的图片就行了

最后更新配置

sudo update-initramfs -u


开机之后的登陆界面的图片

图片加载文件地址：

/etc/alternatives/gdm3.css

修改这个文件中的下面的代码：

#lockDialogGroup {
  background: #2c001e url(resource:///org/gnome/shell/theme/noise-texture.png);
  background-repeat: repeat; }

修改url的加载你的图片位置，修改 background-repeat: 

no-repeat


增加：

background-size: cover;
  background-position: center;

老规矩，重新加载。
