在Ubuntu 18.04桌面系统上安装Wireshark的方法

2019-01-10 15:57:07作者:戴进稿源:云网牛站

本文介绍在Ubuntu 18.04/Ubuntu 18.06桌面系统上安装Wireshark的方法，Wireshark是一个免费的开源网络分析工具和数据包分析器，通过Wireshark，可以实时捕获网络流量，并将其用于网络分析，故障排除，教育等，在Ubuntu 18.04上安装Wireshark可以选择安装稳定Wireshark或开发的版本。对于GNS3用户可以参考在Ubuntu 18.04系统上安装GNS3的方法。

 

安装Stable Wireshark版本

如果你对稳定有要求，而不是尖端功能，那么你可以在Ubuntu 18.04上安装Wireshark的稳定版本。

添加PPA存储库并安装Wireshark：

sudo add-apt-repository ppa:wireshark-dev/stable 

sudo apt update

sudo apt -y install wireshark

 

安装Wireshark开发版

要获得开发版本，请添加：

sudo add-apt-repository ppa:dreibh/ppa

从存储库安装Wireshark：

sudo apt update

sudo apt -y install wireshark

当询问是否允许非超级用户捕获数据包时，请选择你的选项并完成安装：

检查已安装的Wireshark版本：

$ wireshark --version

Wireshark 2.9.0 (Git Rev Unknown from unknown)

 

配置Wireshark

为了能够以普通用户身份捕获数据包，请将你的用户添加到wireshark组：

sudo usermod -a -G wireshark $USER

还要更改dumpcap二进制文件权限：

sudo chgrp wireshark /usr/bin/dumpcap

sudo chmod 750 /usr/bin/dumpcap

sudo setcap cap_net_raw,cap_net_admin=eip /usr/bin/dumpcap

校验：

$ sudo getcap /usr/bin/dumpcap

/usr/bin/dumpcap = cap_net_admin,cap_net_raw+eip

 

启动Wireshark

启动Wireshark应用程序可以从CLI或应用程序启动器完成：

要从GUI启动Wireshark，请搜索wireshark并按Enter键：

通过输入命令可以完成同样的操作：

$ wireshark

要测试数据包捕获，请选择要使用的接口，然后单击开始捕获数据包按钮：

到这里，你已经安装了Wireshark并在你的Ubuntu 18.04台式机上正常运行着。
